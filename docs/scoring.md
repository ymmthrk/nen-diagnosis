# スコアリングモデル

## 回答形式

5段階リッカートスケール（各軸5問 × MAX4pt = 軸MAX 20pt）：

| ラベル | スコア |
|--------|:------:|
| とても当てはまる | 4 |
| やや当てはまる | 3 |
| どちらともいえない | 2 |
| あまり当てはまらない | 1 |
| 全く当てはまらない | 0 |

逆転項目（Q5, Q10, Q15, Q20, Q25）は `4 - rawScore` で変換。

---

## エゴグラム5軸

| 軸 | 名称 | 高い人 | 低い人 |
|----|------|--------|--------|
| CP | 厳格な親 (Critical Parent) | 責任感、正義感、完遂力 | 寛容、おおらか |
| NP | 養育的な親 (Nurturing Parent) | 共感力、世話好き、受容的 | 自立重視、ドライ |
| A  | 合理的な大人 (Adult) | 冷静、論理的、データ重視 | 直感的、即断即決 |
| FC | 自由な子供 (Free Child) | 好奇心、即興的、感情豊か | 慎重、安定志向 |
| AC | 順応する子供 (Adapted Child) | 空気を読む、協調、慎重 | 自己主張、マイペース |

---

## ステップ1：エゴグラム集計

```typescript
axisScore[axis] = Σ(該当5問のスコア)  // 0〜20
```

## ステップ2：エゴグラム → 念系統スコア変換

重み（◎=3, ○=2, △=1, -=0）：

| 念系統 | CP | NP | A | FC | AC | 根拠 |
|--------|:--:|:--:|:-:|:--:|:--:|------|
| 強化系 | 3 | 2 | 0 | 2 | 0 | 責任感+共感+自由さ＝素直で一途 |
| 変化系 | 0 | 0 | 1 | 3 | 1 | 自由さ主導＝気まぐれ |
| 放出系 | 2 | 3 | 0 | 2 | 0 | 共感+責任感+自由さ＝情に厚く大胆 |
| 操作系 | 1 | 0 | 3 | 0 | 1 | 分析力主導＝戦略家 |
| 具現化系 | 1 | 1 | 2 | 0 | 3 | 順応主導＋分析＝慎重で緻密 |

```typescript
nenScore[type] = Σ(weight[type][axis] * axisScore[axis])
```

## ステップ3：MBTI補正

| 傾き | 補正 |
|------|------|
| E | 放出+4, 強化+2 |
| I | 操作+4, 具現化+2 |
| S | 強化+4, 放出+2 |
| N | 変化+4, 特質+2 |
| T | 操作+4, 変化+2 |
| F | 放出+4, 強化+2 |
| J | 具現化+4, 操作+2 |
| P | 変化+4, 放出+2 |

## ステップ4：特質系判定

```typescript
// 一貫性チェック（適当回答フィルタ）
inconsistency = Σ(各軸の |順方向平均 - 逆転項目の反転値|) / 5

if (inconsistency <= 1.5) {
  if (1位スコア - 2位スコア <= 5)    → specialization += 10
  if (エゴグラム5軸の標準偏差 <= 2.0) → specialization += 6
}
```

## ステップ5：最終スコア

```typescript
finalScore[type] = nenScore[type] + mbtiBonus[type] + specializationBonus
topType = argmax(finalScore)
```

## MBTI判定ロジック（MBTIを知らない人向け）

各軸2問、5段階回答。はい寄りの極にスコア加算、逆極に `4 - score` を加算。
合計が大きい方を採用（同点はE/S/T/J優先）。
