### S/N × 念系統

|--------|----------|----------|

| 変化系 | 実在する物質の性質を正確に再現する。キルアの電気のように、現実世界の法則を忠実にオーラへ写し取る。五感の鋭さが変化の精度を決める | 現実には存在しない性質をオーラに与える。ヒソカのバンジーガムのように、「こんな性質があったら面白い」という発想が先にあり、それを実現する |
| 放出系 | 見える距離、体感できる威力で勝負する。目の前の敵に確実にオーラを届ける実践重視。射程や威力を数字で把握し、堅実に運用する | オーラに意味や感情を乗せて遠くまで届ける。センリツの演奏のように、物理的な威力だけでなく、受け手の心に作用する念を放つ |
| 操作系 | 明確なルールと手順で対象を操る。イルミの針のように、具体的な道具と条件を使った確実な操作。再現性と安定性を重視する | 相手の深層心理や行動パターンを読んで操る。目に見えない「流れ」を操作する。将棋のように何手先も読んで盤面を支配する |
| 具現化系 | 実在の物体を精密に再現する具現化。コルトピのギャラリーフェイクのように、現物と寸分違わぬコピーを作る。細部への執着が精度を生む | 概念や抽象物を具現化する。クラピカの「律する小指の鎖」のように、「誓い」「制約」といった目に見えないものに形を与える。理念が先、形は後 |
| 特質系 | 五感を超越した知覚が開花する。パクノダのように触れるだけで記憶を読み取るなど、通常の感覚の延長線上にありながら次元が違う | 既存の念の法則そのものを書き換える力。クロロのスキルハンターのように、システムのメタレベルで作用する。本人にすら全容が把握できない可能性がある |

### T/F × 念系統

| 念系統 | T（思考） | F（感情） |
|--------|----------|----------|
| 強化系 | 効率を追求した強化。無駄な動きを削ぎ落とし、最小のオーラで最大の効果を出す合理的な戦闘スタイル。訓練量と数値で実力を測る | 感情が昂るほどオーラが増幅する。仲間への想い、敵への怒り——心の動きがそのまま出力に直結する。ゴンの「もうこれで終わってもいい」が最大の強化 |
| 変化系 | 変化の条件を論理的に設定する。「温度をN度に変える」「粘度をこの範囲に保つ」など、パラメータを数値管理する精密な変化系 | 感情の色がオーラの性質に反映される。怒りなら炎、悲しみなら冷気のように、そのときの心理状態が変化の方向性を決める。美的感覚が能力を形作る |
| 放出系 | 計算された弾道と最適な射程で運用する。無駄打ちしない、感情に流されない。フランクリンの「俺の弾に当たったやつが悪い」的な淡々とした運用 | 感情を込めるほど遠く、強く届く。怒りのパンチ、守りたい気持ちのバリア——放出されるオーラに感情の温度がそのまま乗る |
| 操作系 | ルールベースの操作。条件Aなら行動B、という明確なプログラムを対象に埋め込む。バグがなく、感情に左右されない精密な制御 | 対象との心理的な繋がりを通じて操作する。信頼や恐怖といった感情的な紐帯が操作の媒介になる。共感力の高さが操作の精度に直結する |
| 具現化系 | 機能性を優先した具現化。クラピカのダウジングチェーンのように、明確な目的のために最適化された道具を作る。美しさより実用性 | 美学やこだわりが形になる具現化。見た目の完成度、素材の質感、使い心地——機能だけでは説明できない「好き」が詰まった一品を生み出す |
| 特質系 | 能力のルールを俯瞰的に把握し、戦略的に運用する。クロロのように能力の組み合わせを知的パズルとして楽しむ | 他者の感情を感知・操作する共感型の特質系。ネオンの占いのように、意図せず発動する直感的な力。感情の受信機として機能する |

| 変化系 | 変化パターンをあらかじめ複数用意しておく。状況に応じて切り替える「型」を持つ。準備した変化ほど精度が高い | その瞬間の閃きでオーラの性質を変える即興型。ヒソカのように、戦いの中で最も面白い変化を選び取る。予測不能さが最大の武器 |
| 放出系 | 射程距離、威力、消費オーラを事前に計算して運用する。計画通りの射撃が最も効率的。待ち伏せや罠との組み合わせが得意 | 感覚と勢いで撃つ。距離も威力もそのときの感情次第。型にはまらないからこそ、敵の予測を超える一撃が生まれる |

| 具現化系 | 設計図を完璧に描いてから具現化する。クラピカが修行で鎖の質感まで記憶したように、事前準備の精度が具現化の完成度を決める | 必要になった瞬間に「こういう道具が欲しい」と思って具現化する。完璧さより即応性。状況対応型の道具を次々生み出す |
| 特質系 | 自分の能力のルールを把握し、制約と誓約で体系化する。能力の発動条件を自ら設計することで最大効率を引き出す | 能力の全容を自分でも把握しきれていない。新しい状況で未知の力が覚醒する可能性がある。制約より発見。成長の余白が大きい |
